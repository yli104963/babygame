## 仓库名称

`babygame`

## 仓库短描述

一个面向低龄儿童的轻量小游戏系统，包含 23 个小游戏、成长奖励、今日任务、家长模式和离线缓存能力。

> 彩虹游乐园是一个围绕低龄儿童亲子场景设计的轻量小游戏系统。它不是单个小游戏页面，而是一整套可以直接打开、直接游玩、并带有成长反馈与家长控制能力的儿童互动站点。

## 推荐上传的截图

2. `game-layout.png`
   主游戏区截图，展示中间内容和两侧菜单布局

## 截图说明文案

### 首页截图说明

> 首页提供精选入口、推荐路线、最近玩过和成长模块，减少第一次进入时的选择压力。

### 主游戏区截图说明

> 游戏主界面针对大屏和小屏做了不同菜单布局，保证内容区优先、导航不打扰。

### 家长模式截图说明

> 家长模式支持档案切换、外观主题、难度、声音、提醒和内容显示范围控制。

## 发布后的推荐操作

1. 上传代码到 GitHub
2. 在仓库 About 中补 Description 和 Topics
3. 添加 3 张项目截图
4. 开启 GitHub Pages
5. 把 Pages 地址填回仓库 Website
