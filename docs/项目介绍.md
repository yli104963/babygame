# 项目介绍

## 这是什么

彩虹游乐园是一个面向低龄儿童的轻量小游戏系统。

它不是单个小游戏，而是一整套可以直接打开使用的儿童互动站点。孩子进入后可以快速开始游戏，家长也能通过家长模式控制难度、声音、提醒与可见内容。

## 想解决的问题

很多儿童小游戏页面会有几个典型问题：

- 打开后选择太多，孩子不知道先玩哪个
- 只有游戏，没有连续体验
- 缺少家长控制和成长反馈
- 页面花哨但不好用
- 手机上能打开，但不好操作

彩虹游乐园希望把这些问题一起解决。

## 项目的核心思路

### 1. 打开就能玩

首页会优先给出精选入口、推荐路线、最近玩过和快速筛选，尽量减少第一次使用时的选择压力。

### 2. 游戏之外也有成长感

系统不仅有小游戏，还加入了：

- 勇气星星
- 每日任务
- 徽章墙
- 连续打卡
- 小伙伴收集
- 贴纸收集

这样孩子每次回来都有“继续成长”的感觉。

### 3. 家长可控

家长模式里可以管理：

- 孩子档案
- 外观主题
- 难度
- 声音
- 护眼提醒
- 游戏显示范围

这让它更像一个完整的小系统，而不是一页散乱小游戏。

### 4. 多端可用

项目采用纯前端静态实现，手机、平板、电脑都能打开；同时加入了 PWA 和离线缓存能力，适合做桌面快捷入口或主屏图标。

## 适合谁

- 想给孩子准备一个轻量互动站点的家长
- 想做儿童教育产品原型的设计 / 前端开发者
- 想把多个儿童小游戏整合成一个系统的人

## 当前内容覆盖

项目当前覆盖以下内容方向：

- 反应类
- 记忆类
- 认知类
- 观察类
- 启蒙类
- 节奏类
- 思维类
- 识字类
- 英语类
- 描线练习类

## 产品气质

这个项目强调的不是“炫”，而是：

- 亲和
- 直接
- 清楚
- 低压
- 有连续体验

它更像一个可以一直慢慢完善的儿童互动产品雏形。
