# 彩虹游乐园

一个打开就能玩的儿童小游戏合集，面向低龄儿童设计，支持手机和电脑访问。

## 项目亮点

- 23 个儿童小游戏整合在一个统一系统里
- 首页有精选入口、推荐路线、最近玩过和快速筛选
- 内置勇气星星、今日任务、徽章墙、连续打卡、伙伴与贴纸收集
- 提供家长模式，可管理档案、主题、难度、声音和护眼提醒
- 纯前端静态实现，可直接部署到 GitHub Pages
- 支持本地存档与离线缓存，适合长期迭代成儿童产品原型

## 仓库首页介绍

彩虹游乐园是一个围绕低龄儿童亲子场景设计的轻量小游戏系统。它不是单个小游戏页面，而是一整套可以直接打开、直接游玩、并带有成长反馈与家长控制能力的儿童互动站点。

项目目前已经包含反应、记忆、认知、观察、启蒙、节奏、思维、识字、英语和描线等多个方向的小游戏，同时补充了儿童产品常见的成长系统模块，让体验不只停留在“玩一下”，而是更接近一个完整的小型儿童产品。

## 项目简介

彩虹游乐园是一个纯前端静态小游戏站点，围绕“轻量、直接打开就能玩、适合亲子场景”来设计。项目内置 23 个小游戏，包含反应、记忆、认知、观察、启蒙、节奏、思维、识字、英语和描线等内容。

除了游戏本身，项目还补充了儿童产品常见的成长体验模块，比如：

- 勇气星星奖励
- 今日挑战
- 徽章墙
- 连续打卡
- 小伙伴与贴纸收集
- 家长模式
- 本地进度保存
- PWA / 离线缓存支持

## 本地运行

### 方式一：直接双击

双击项目目录中的：

```text
打开彩虹游乐园.command
```

它会自动启动本地服务并打开页面。

### 方式二：命令行启动

```bash
cd "/Users/god/Desktop/儿童小游戏"
python3 -m http.server 4174 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:4174/index.html
```

## 部署到 GitHub Pages

这个项目是静态站点，上传到 GitHub 后可以直接发布到 GitHub Pages。

基础流程：

1. 创建一个 GitHub 仓库
2. 把本项目推送到该仓库
3. 在 GitHub 仓库设置中开启 Pages
4. 选择从默认分支部署
